# Non-Implementation Boundary

This repository is a protected public proof surface.

It excludes:

```text
private runtime law
protected enforcement internals
fabric-level proprietary thresholds
customer-specific corridors
customer policies
production adapters
private evaluator algorithms
receipt-generation internals
replay-verification internals
protected SMC hardware implementation
deployment-sensitive architecture
```

The repository provides public proof framing and a software-adjacent validator only.

## Claim discipline

This repository does not claim:

```text
NVIDIA partnership
NVIDIA certification
CUDA-level enforcement
kernel-level enforcement
fabricated SMC hardware
production deployment
```

It defines a pilot-ready accelerator-boundary proof corridor and candidate integration surfaces.
